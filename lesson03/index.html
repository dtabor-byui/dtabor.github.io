Site Purpose :


The purpose of this site is to provide weather information and basic data for each of the listed towns. It is also designed to contain other data such as local activities, tourist attractions, current growing conditions, any storm warnings for each community. This information should allow users to make informed decisions about their outdoor activities and other events such as crop harvest and planting times for more sensitive vegetables, as well as give indications for river hatches for fishing and basic lunar information.
